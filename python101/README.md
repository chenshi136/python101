数据清洗核心规则:
1去除干扰
噪音与非文本过滤去除[图片]、[红包]等系统消息;去除URL链接;去除乱码与非常规特殊符号，仅保留常规标点和Emoii
2碎片信息合并
设定时间闻值(如60秒)，将同一人在此期间内发送的、未被中断的连续消息合并。例如，将"其实吧”>“我觉得"->“这就很难评"合并为"其实吧，我觉得这就很难评
3隐私脱敏
使用正则表达式或手动方式，将真实姓名、手机号、具体地址等替换为标准化标签，如<name>,<phoneE>,<address>。
4上下文修正
对干包含强烈指代词(如“对他就是那样”)、但脱离上下文后含义模糊的句子，直接删除，优先保留独立成句、观点清晰的语料。
5人称代号统一
将同一个人的不同称呼做诠释，避免分辨不清人物，保持语句对象明确性。对代词做出指代对象诠释。
6检查错字
避免数据中出现错字导致ai理解失误从而某个概念认知不清晰。
7引入事件去除
将突然插入的聊天内容进行补全或者去除，避免无厘头对话出现。
8量词明确
对量词进行标准处理，如“一个水”改为“一杯水”或者“一瓶水”
